
# 🌤️ Skybound Adventurer

A fun and fast-paced **2D platformer game** built using **HTML5 Canvas, CSS, and JavaScript**. Jump across platforms, defeat enemies, collect coins, and reach the flag to win!

---

## 👨‍💻 Author

**Keshav Jha**

---

## 🎮 Game Overview

**Skybound Adventurer** is a side-scrolling platformer where the player navigates through obstacles, avoids falling, defeats enemies, and collects coins to increase score.

---

## ✨ Features

* 🧍 Smooth player movement with physics (gravity, friction)
* 🪙 Coin collection system with score tracking
* 👾 Enemy AI with patrol behavior
* 💥 Particle effects for actions (jump, enemy defeat, coins)
* ❤️ Lives system with respawn mechanics
* 🏁 Victory condition (reach the flag)
* 💀 Game Over screen
* 📱 Mobile-friendly controls (touch support)
* 🎨 Stylish UI with animations and gradients

---

## 🕹️ Controls

### 💻 Desktop

| Key       | Action            |

| ← / →     | Move Left / Right |

| R         | Restart Game      |

### 📱 Mobile

* On-screen buttons for movement and jumping

---

## 🚀 How to Run

1. Download or clone the project
2. Open the `index.html` file in any modern browser
3. Click **"PLAY NOW"** and start playing!

---

## 🧠 Game Mechanics

* **Jumping:** Only possible when grounded
* **Enemies:**

  * Jump on top → defeat enemy
  * Touch from side → lose a life
* **Falling off map:** Instant death
* **Coins:** +10 points each
* **Enemies defeated:** +50 points
* **Goal:** Reach the flag to win
